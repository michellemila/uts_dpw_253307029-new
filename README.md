**Deskripsi :** 
Project ini merupakan sebuah website portal pendaftaran siswa baru (PPDB) yang dibuat untuk membantu calon siswa melakukan pendaftaran secara online. Website ini dirancang dengan tampilan modern agar terlihat menarik dan nyaman digunakan.

Website ini terdiri dari tiga halaman utama, yaitu halaman beranda (index.html), halaman alur pendaftaran (alur.html), dan halaman formulir pendaftaran (form.html). Halaman beranda berisi informasi umum serta tabel jadwal pelaksanaan PPDB. Halaman alur pendaftaran menampilkan langkah-langkah proses pendaftaran yang disertai gambar ilustrasi. Sedangkan halaman formulir digunakan untuk mengisi data pendaftaran seperti nama lengkap, NIK, dan jalur pendaftaran.

Tampilan website dibuat menggunakan CSS eksternal agar terlihat lebih rapi dan modern, dengan tambahan efek seperti gradasi warna, animasi hover, dan tampilan form yang interaktif. Selain itu, digunakan JavaScript untuk memberikan validasi pada form, sehingga data yang dimasukkan harus lengkap dan sesuai ketentuan, seperti NIK yang harus berjumlah 16 digit.
